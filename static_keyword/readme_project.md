# Java Static Keyword Mini Project

## 📌 Project Description

This is a small Java project created to demonstrate the use of the **static keyword** in Java.
The project simulates a simple college system where student details are stored and the total number of students is counted using a static variable.

## 🎯 Concepts Covered

This project demonstrates the following Java concepts:

* Static Block
* Static Variable
* Static Method
* Constructor
* Object Creation
* Instance Method

## 🏫 Program Behavior

1. A **static block** displays a welcome message when the class is loaded.
2. A **static variable** (`no_of_student`) keeps track of the total number of students.
3. Each time a new student object is created, the student count increases.
4. A **static method** prints the total number of students.
5. An **instance method** displays individual student details.

1. Compile the Java files

```

```

